# VK Бот-напоминалка на Go

Бот для ВКонтакте, который позволяет создавать задачи с напоминанием. Данные хранятся в PostgreSQL.

## Требования
- Go 1.21+
- PostgreSQL

## Настройка
1. Скопируйте `.env.example` в `.env` и заполните реальными данными.
2. Создайте базу данных PostgreSQL.
3. Получите токен сообщества ВКонтакте (раздел «Работа с API»).

## Запуск
```bash
go mod tidy
go run main.go

```

## Команды
```bash
/add [текст] ГГГГ-ММ-ДД ЧЧ:ММ  # добавить задачу
/list                           # показать задачи
```
